# FlowSynx Plugins API

The FlowSynx Plugins API allows clients to manage plugin lifecycle operations such as 
listing, installing, retrieving details, updating, and removing plugins. 
It uses RESTful principles and supports both **Basic Authentication** and **Bearer Token Authentication**.

## Authentication
### 1. Basic Auth

```bash
curl -u username:password http://localhost:6262/plugins
```

### 2.Bearer Token
```bash
curl -H "Authorization: Bearer <your_token>" http://localhost:6262/plugins
```

## API Endpoints
### List Installed Plugins
- **Method**: `GET`
- **Endpoint**: `/plugins`
- **Body Required**: Yes

#### `curl` Example
```bash
curl -X GET http://localhost:6262/plugins \
  -H "Authorization: Bearer <your_token>"
```

#### Example response
```json
{
  "data": [
    {
      "id": "6c60c794-99b9-47e8-aa34-18a14cdc4583",
      "type": "FlowSynx.Execution.ExternalProcess",
      "version": "1.0.0",
      "description": "Execute external processes, scripts, or commands and capture their output, enabling integration of external tasks within workflows."
    }
  ],
  "messages": [],
  "succeeded": true,
  "generatedAtUtc": "2025-08-02T13:10:05.2517196Z"
}
```

### Install Plugin
- **Method**: POST
- **Endpoint**: /plugins
- **Body Required**: Yes

#### Request Body
```json
{
  "Type": "FlowSynx.Execution.ExternalProcess",
  "Version": "1.0.0"
}
```

#### `curl` Example
```bash
curl -X POST http://localhost:6262/plugins \
  -H "Authorization: Bearer <your_token>" \
  -H "Content-Type: application/json" \
  -d '{
        "Type": "FlowSynx.Execution.ExternalProcess",
        "Version": "1.0.0"
      }'
```

#### Example response
```json
{
  "data": {},
  "messages": [
    "The plugin has been installed successfully."
  ],
  "succeeded": true,
  "generatedAtUtc": "2025-08-02T13:09:52.9549804Z"
}
```

    "specifications": []

  "messages": [],

### Update Plugin
- **Method**: PUT
- **Endpoint**: /plugins
- **Body Required**: Yes

#### Request Body
```json
{
  "Type": "FlowSynx.Execution.ExternalProcess",
  "Version": "1.0.0"
}
```

#### `curl` Example
```bash
curl -X PUT http:/localhost:6262/plugins \
  -H "Authorization: Bearer <your_token>" \
  -H "Content-Type: application/json" \
  -d '{
        "Type": "FlowSynx.Execution.ExternalProcess",
        "Version": "1.0.0"
      }'
```

#### Example response
```json
{
  "data": {},
  "messages": [
    "The plugin has been uninstalled successfully."
  ],
  "succeeded": true,
  "generatedAtUtc": "2025-08-02T13:09:52.9549804Z"
}

### Uninstall Plugin
- **Method**: DELETE
- **Endpoint**: /plugins
- **Body Required**: Yes

#### Request Body
```json
{
  "Type": "FlowSynx.Execution.ExternalProcess",
  "Version": "1.0.0"
}
```

#### `curl` Example
```bash
curl -X DELETE http://localhost:6262/plugins \
  -H "Authorization: Bearer <your_token>" \
  -H "Content-Type: application/json" \
  -d '{
        "Type": "FlowSynx.Execution.ExternalProcess",
        "Version": "1.0.0"
      }'
```

#### Example response
```json
{
  "data": {},
  "messages": [
    "The plugin has been uninstalled successfully."
  ],
  "succeeded": true,
  "generatedAtUtc": "2025-08-02T13:09:52.9549804Z"
}
```
