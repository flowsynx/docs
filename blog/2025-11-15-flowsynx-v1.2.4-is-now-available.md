## FlowSynx v1.2.4 is officially released!

We're excited to roll out version 1.2.4 of FlowSynx â€” our lightweight, secure, and plugin-driven workflow engine. 

This is the v1.2.4 release of flowsynx

## Features
- Add AI-driven workflow generation and optimization ([#691](https://github.com/flowsynx/flowsynx/issues/691))
- Add support for conditional and arithmetic parsing ([#692](https://github.com/flowsynx/flowsynx/issues/692))
- Add support for functional methods in ExpressionParser ([#694](https://github.com/flowsynx/flowsynx/issues/694))
- Add notification system with email provider support ([#709](https://github.com/flowsynx/flowsynx/issues/709))
- Introduced `WebhookNotificationProvider` to enable sending notifications via webhooks ([#712](https://github.com/flowsynx/flowsynx/issues/712))
- Add DisabledAuthenticationProvider for disabled auth mode ([#713](https://github.com/flowsynx/flowsynx/issues/713))
